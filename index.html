Virtual Fitting Rooms For Online shopping :

The Semester Exams database is designed to store information about student exams. The database includes several entities such as Student, Invigilator, Exam, Subject, Department, Exam Room, and Exam Seat. Students are identified by their unique Student ID and are associated with a Department. Each Department can offer multiple Subjects. Each Exam is associated with a Subject and is held in an Exam Room. Each Exam Room is associated with a Block.

SVEC Map-PIN :

"SVCE Maps-Pin" is a web-based solution designed to assist students in locating exam rooms within large college campuses. By entering their student ID, students gain access to personalized information such as roll numbers, block names, classroom numbers, and exam instructions, alleviating the stress of finding exam rooms during exam periods. The platform also provides navigational directions from the college's main entrance to the exam room. Implementation involves database setup, web development, user authentication, and stringent security measures. "SVCE Maps-Pin" aims to enhance students' exam experiences, reduce tardiness, and optimize exam time utilization, fostering an efficient and stress-free academic environment.

Design of wardrobe using stacks:

This program designs a wardrobe using the concept of stacks in C programming language. The objective is to allow for easy access to the dresses kept inside the wardrobe, with the dress kept last being the first to be taken out. The implementation of stacks allows for efficient storage and retrieval of the dresses, ensuring that the wardrobe remains organized and clutter-free. The program provides a simple and efficient solution for managing a wardrobe, making it an ideal choice for those who value organization and ease of use.
